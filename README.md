# Images

| A small tool to help you make the right decision when **choosing image format, size and quality** (compression ratio).

## 🏁 Run locally

```
$ yarn
$ yarn start
```

## 🖼 Re-generate images

You'll need [ImageMagick](https://imagemagick.org/index.php) installed.

```
$ yarn regenerate
```

## 🚀 Deploy

```
$ yarn deploy
```
